# AZ AI Ingestion Processor (Experimental)

## Quick Start

### Run the tests

```bash
uv run pytest
```

### Run the examples

```bash
uv run examples/itsarag.py
uv run examples/argus.py
```

> [!NOTE]  
> `itsarag.py` also generates a Mermaid diagram of its graph in [examples/itsarag.md](examples/itsarag.md).
