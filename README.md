# AZ AI Ingestion Processor (Experimental)

## Quick Start

### Run the tests

```bash
uv run pytest
```

### Run the examples

```bash
uv run examples/its_a_rag.py
uv run examples/argus.py
```

> [!NOTE]  
> `its_a_rag.py` also generates a Mermaid diagram of its graph in [examples/its_a_rag.md](examples/its_a_rag.md).
